# magicMirror

魔镜魔镜，无所不知的魔镜[-_-] //并不是

### 安装
```cmd
git clone https://github.com/memirror/magicMirror.git

cd magicMirror

python setup.py sdist
pip install -e .
```


### 初始化数据

```cmd

python script/start.py
```

### 启动
```cmd

python magicMirror

or

# pip install 完毕后
magicmirror    
```
